# GridSentinel Multi-Tool Simulation

An interactive simulation dashboard for hybrid microgrid analysis.

## Features

- HOMER Pro Interface
- MATLAB/Simulink Dashboard
- Tinkercad Circuit
- KiCad
- LTSpice
- OrCAD/PSpice
- Autodesk Eagle
- RETScreen

## Technologies

- HTML
- CSS
- JavaScript
